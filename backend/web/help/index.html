User Manual

Introduction

This user manual will guide you through the functionalities of the Re-member administrator module. Please use the menu on the left to choose an appropriate topic of your choice and then scroll down on the contents listed to see how to perform a particular task and successfully navigate around the Re-member administrator module.


The administrator module/admin application is directly connected to the mobile app and all the changes made on the admin application is reflected on the mobile app.
