User Manual

Committee

How do I create a new Committee?

1. Committee Page

From the "Main Menu" select "Committee" to go to the committee page

2. Name of the committee

Start by naming the committee. Type in the appropriate name for the committee and continue on by clicking the "Add" button and the new committee name will be listed below.

3. Edit Committee

You can edit an existing or a newly created committee name by clicking on the "Edit" button and the committee name will be displayed on the textbox above. From which you can rename the committee and click on the "Update" button to confirm updation.

4. Create Designation

Add a new designation to the committee by typing in a new name of the required designation and click on the "Add" button. The new designation will be listed below and you can also edit the designation name the same way the committee name is edited, by clicking on the "Edit" button beside the designation title.

5. Set committee period

After adding the designations to a committee, set the time period for this committee by selecting the from the downdown, the name of the committee and the date period from the fields provided.

How do I add a member to the Committee?

1. Committee Page

From the "Main Menu" select "Committee" to go to the committee page

2. Adding a member to committee

Search for a member by name on the "Find Member" tab and set the designation and the period of committee membership and click "save" to add the selected member as a committee member

How do I search for a committee member?

1. Go to Committee

Click on the committee icon from the dashboard to navigate to the committee page

2. Search for a committee member

On the "Find Members" tab search by the name of the member on the text box provided and click search

How do I add the spouse of a member as a committee member?

1. Adding a member spouse to committee

Search for a member spouse by name on the "Find Member" tab while keeping the "spouse" checkbox checked. Once the spouse name is found, set the designation and the period of committee membership and click "save" to add the selected spouse as a committee member

How do I remove a member from the Committee?

1. Remove a committee member

Click on the "Committee Member" tab on the committee page and sort members by period, then select the member to be removed from the list and click "Deactivate"
