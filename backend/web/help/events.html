User Manual

Events

How do I create an event?

1. Event Registration

Click on the "Event Registration" link and you will be redirected to the event registration page

2. Update event title and description

Use this form to update the name of an event as well as give a brief description of the event.

3. Fill in the details

The details like venue, resource, day chair/champion etc are to be filled in here

4. Date and time

Event date and time : The date of the event should be set as shown below

Activate on : The date from which the event will be shown in the app

Expiry date : The date on which the event will be removed from the app

5. How to set date and time

Click on the date input box and select the date, then a time (Hour) and then minutes as shown. The date and time of the event is now set

6. Save

The details like venue, time and date etc are to be filled in here. Then click on the "Save" button at the bottom.

7. Event created alert

After clicking the save button, an alert will show up confirming the creation of the event. Click "Okay" to continue and now you have successfully created an event

8. Publish Event

Go back to the event listing page and click on the "Publish" button to push this event into the app immediately. Else, the event will only show up on the "activation date" set while creating the event

How do I search for an event?

1. Go to Events page

Click on the Event icon from the dashboard to navigate to the Events listing page

2. Search Event Listing

Search for an event by title or date on the text field provided on top

How do I edit an event?

1. Select Event

Select the event from the event listing screen of which you need to edit and click on the "Manage" button. It will open the edit screen

2. Edit the title and content

Type in the title of the event and the any text content describing the event as such using the rich text editor

3. Fill in the details

The details like venue, time and date etc are to be filled in here. Then click on the "Save" button at the bottom.

You have now edited an event

How do I add a graphic image or flyer for an Event?

1. Go to event registration

To add an image into a new event, click on the "Event Registration" menu item on the drop down of the events link on the main menu

2. Adding an image

Click on the image icon on the event registration page and a popup will appear

3. Upload a picture/graphic image

Choose an image by clicking the "Choose file/browse" button and then click on the "Send it to the server" button

4. Set image info

The pop switches to the image info tab and you have to set the image width as 100% and leave the height field as blank and click "Okay".

You have now added an image into the event

How do I set RSVP for an event?

1. Setting RSVP

On the "Event Registration" page there is an option to set RSVP. If "Yes" is selected the member will see a "Would you like to attend" option in the app.

You can use the "RSVP Listing" report to see how many members are likely to attend a function.

2. Confirm RSVP

Click on the "Save" button to cofirm RSVP is to be activated

How will I know how many members are planning to attend an Event?

1. Select Event

Select the event from the event listing screen of which you need to know the number of attendees and click on the "RSVP Listing" button

2. RSVP Listing

A table will display all the members, children and guests count
