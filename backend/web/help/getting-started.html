Getting Started
Get started with the login screen. A username and password is requested here and inputing the correct credentials will grant access to the "Dashboard".

The dashboard is the starting point of the web application. You can access all pages from this point on.
Dashboard

The dashboard lets the user access the following menu items instantly :

  • Institution
  • Members
  • Events
  • News
  • Bills
  • Admin Profile
  • Feedback
  • Committee
  • Conversations
