User Manual

Feedback

How do I view feedbacks?

1. Go to Feedback page

Click on the "Feedback" link on the main menu to navigate to the Feedback page

2. Search for a feedback

On the feedback tab, select the type of feedback, the date ranges and the rating to find a specific feedback post

How do I reply to a feedback ?

1. Reply

On the feedback tab, click on any feedback title to expand and view the details and photos (if any). You can also reply to the feedback on the text fields provided under. Then click on the "Respond" button to send the response back.

How do I change the feedback settings ?

1. Add feedback type

On the settings tab, you can add a new feedback type to the the existing list by entering the feedback type to the text field provided and click on the "Add" button.

2. Rearrange feedback type

You can rearrange the feedback options using the up (green arrow) and down (red arrow) arrows. Click it once and move the feedback type coresponding to the arrow, up or down.

3. Feedback email

Type in the email addresses separated by a semi colon (;) to which the acknowledgment has to be sent to
