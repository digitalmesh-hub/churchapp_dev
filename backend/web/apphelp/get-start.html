User Manual

Getting Started

Get started with the login screen. Your mobile number is requested here. After inputting the number click on the LOGIN button, then a 4 digit OTP Will be sent to the corresponding mobile number.
After clicking on the LOGIN button you will be directed to a page where you have to enter the 4 digit OTP which was sent earlier.

After entering the correct 4 digit OTP you will be granted access to the "Dashboard".

The dashboard is the starting point of the Mobile application. You can access all pages from this point on.

Dashboard

The dashboard lets the user access the following menu items instantly :

  • Notification
