User Manual

Profile

How do I view my own profile ?

1. Go to Profile page

Click on the user icon in the bottom of the page to navigate to the Profile page

You will be navigated to another page where you can view your complete profile

How do I edit my profile ?

1. Go to Profile page

Click on the edit icon in the profile page

You will be directed to a page where you can edit your complete profile

How do I edit my spouse details ?

1. Go to Profile page

Click on the edit icon on the spouse detail

You will be directed to a page where you can edit you spouse details

How do I add or edit my dependants ?

1. Go to Profile page

Click on the edit icon in the profile page

2.Select dependants tab

On the edit screen, click on the dependant tab on the top. You can add or edit dependant details on that screen
